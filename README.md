# 📊 Retail Sales Data Analysis & Visualization

## 📖 Overview

This project focuses on analyzing and visualizing retail sales data using Python. The goal is to extract meaningful insights from the dataset through data analysis and graphical visualization techniques.

---

## 🚀 Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

---

## 📌 Project Features

- Load and explore retail sales dataset
- Data cleaning and preprocessing
- Sales analysis
- Profit analysis
- Regional sales analysis
- Data visualization using different charts
- Generate meaningful business insights

---

## 📊 Visualizations Included

- Line Charts
- Bar Charts
- Pie Charts
- Histograms
- Scatter Plots
- Seaborn Visualizations

---

## 📂 Files Included

- `Retail Sales Data Analysis & Visualization.ipynb`
- `retail_sales.csv`
